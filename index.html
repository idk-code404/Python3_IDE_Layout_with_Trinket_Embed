Python Trinket 3 Embed with Packages

Below is an embedded Python Trinket project. You can interact with it directly in your browser.

Adding Packages

Trinket supports many common Python packages out-of-the-box. To add one, click the **gear icon ⚙️** in the Trinket editor, choose Packages, and search for the library you need.
